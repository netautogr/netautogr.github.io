αbout

Επειδή η μικρή γωνιά που φτιάχτηκε στο Slack για τους network automation engineers “ξεχνάει” εύκολα, δημιουργήθηκε ένα GitHub account με σκοπό την αποθήκευση της γνώσης που κατά καιρούς περνάει από εκεί. Η παρούσα σελίδα είναι μια απόπειρα να ταξινομηθούν τα παραπάνω…

English

A slack group has been created to host our discussion group for Greek Network Engineers (and more) interested in Network Automation, but since slack doesn’t store posts permanently, this github pages site has been created to host our articles and lists of resouces so all those posts can be archived and be kept available..

It’s mostly in Greek but if something catches your eye that interests you and want to contact us you can contact one of our admins on twitter:
